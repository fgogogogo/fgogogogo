粒子雨

向下滚动页面，观察粒子密度随滚动距离动态变化。使用右上角控制面板调节速度、密度、长度和颜色主题。

关于粒子雨

粒子雨是一种经典的视觉效果，通过在 Canvas 上绘制大量下落的粒子线条来模拟雨滴。本项目使用原生 Canvas 2D API 实现，通过对象池复用和增量绘制技术确保高帧率运行。

技术实现

采用固定大小粒子池避免频繁的内存分配与垃圾回收。每帧只清除整个画布一次，批量绘制同色粒子以减少上下文状态切换。滚动监听使用 requestAnimationFrame 节流，避免布局抖动。

粒子颜色通过 HSL 色彩空间生成，在同一色相范围内做微小随机偏移，既保持视觉统一又富有变化。切换主题时新旧粒子自然过渡，无需重置整个粒子池。

性能优化

对象池设计：预分配固定数量的粒子对象，运行时只修改属性而不创建新对象。批量绘制：按颜色分组一次 stroke 调用绘制多条线段。视图裁剪：只绘制视口内可见的粒子。使用 performance.now() 做帧间时间差计算，确保不同刷新率设备上动画速度一致。

交互设计

密度随滚动距离线性增加，在页面底部达到基础密度的 2.5 倍。速度、密度、长度三个参数均可通过滑块实时调节，立即生效无需刷新。预设颜色主题一键切换，覆盖赛博青、极光绿、樱花粉、烈焰橙、电光蓝五种风格。

滚动到底部试试

继续向下滚动，粒子会越来越密集，营造出暴风雨般的效果。到达页面底部时粒子密度达到峰值，形成壮观的视觉冲击。

适用场景

个人博客背景、活动现场大屏、产品发布页面、赛博朋克风格网站等。粒子雨效果轻量且富有科技感，适合各种需要动态背景的前端项目。

所有代码均在一个 HTML 文件中实现，无外部依赖，可直接嵌入任何网页使用。

细节打磨

每颗粒子的透明度、宽度和速度均带有随机扰动，避免整齐划一的机械感。粒子池在滚动导致密度变化时动态增减活跃粒子数量，而非重建池子。

画布尺寸随窗口 resize 自动适配，设备像素比（DPR）正确处理，在高分屏上保持清晰。
