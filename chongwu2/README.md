# 🐾 宠物饮水量数据可视化工具

一个清新风格的宠物饮水量追踪应用，帮助宠物主人记录和可视化宠物的7天饮水量数据。

## ✨ 功能特点

- 📊 **7天数据可视化** - 直观的柱状图展示近一周饮水趋势
- 💧 **今日饮水记录** - 快速记录每次饮水量，支持快捷添加
- 📈 **智能统计** - 自动计算7天平均、最高记录、达标天数
- 🎯 **饮水建议** - 根据宠物体重自动计算建议饮水量
- 🐱 **多宠物支持** - 支持猫咪、狗狗等多种宠物类型
- 📝 **历史记录** - 查看最近的饮水记录
- 💾 **本地存储** - 数据保存在浏览器中，不丢失

## 🚀 使用方法

### 方法一：直接打开
1. 双击 `index.html` 文件即可在浏览器中打开使用

### 方法二：本地服务器
```bash
# 使用 Python 启动本地服务器
python -m http.server 8000

# 或使用 Node.js 的 http-server
npx http-server
```

然后在浏览器中访问 `http://localhost:8000`

## 📱 功能指南

### 添加饮水记录
1. 在「本次添加水量」输入框中输入水量（ml）
2. 点击「添加记录」按钮或按回车键
3. 或使用快捷按钮（+50ml、+100ml等）

### 设置宠物信息
1. 点击宠物名字可以修改
2. 输入宠物体重会自动更新建议饮水量
3. 选择宠物类型（猫咪/狗狗/其他）

### 查看数据统计
- **今日总计**：当天的总饮水量
- **进度条**：显示今日饮水量占建议量的百分比
- **7天趋势图**：查看一周的饮水变化
- **统计数据**：7天平均、最高记录、达标天数

## 💡 技术栈

- **HTML5** - 页面结构

- **JavaScript (ES6+)** - 交互逻辑
- **Chart.js** - 数据可视化
- **LocalStorage** - 数据持久化

## 🎯 饮水建议标准

- **猫咪**：约 60ml/kg 体重/天
- **狗狗**：约 75ml/kg 体重/天

## 📌 数据说明

- 所有数据存储在浏览器本地，不会上传到服务器
- 清除浏览器缓存可能会导致数据丢失
- 建议定期截图保存重要数据
