交互式通知系统

Modern Web Notification System with Smooth Animations

通知类型

// 显示不同类型的通知
// 成功 - 绿色
notify.success('操作成功完成', '保存成功');

// 错误 - 红色
notify.error('请检查您的网络连接', '连接失败');

// 警告 - 橙色
notify.warning('磁盘空间即将不足', '系统警告');

// 信息 - 蓝色
notify.info('新版本可用', '更新提示');

优先级系统

高优先级通知在屏幕上停留的时间更长。鼠标悬停可暂停倒计时。 

// 不同优先级的停留时间
notify.show({
  type: 'error',
  title: '严重错误',
  message: '系统遇到严重问题',
  priority: 'critical'  // 停留15秒
});

堆叠效果

多个通知会自动堆叠，最多显示5个。点击任意通知可手动关闭。 

// 批量生成通知
for (let i = 1; i <= 5; i++) {
  setTimeout(() => {
    notify.info(`通知 #${i}`, '堆叠演示');
  }, i * 300);
}

// 关闭所有通知
notify.closeAll();

功能特性

🎬 流畅动画

使用 CSS3 动画和 JavaScript 配合，实现从右侧滑入滑出的流畅效果

🎨 类型颜色

成功(绿)、错误(红)、警告(橙)、信息(蓝)四种类型，每种都有独特的视觉设计

⏱️ 智能计时

鼠标悬停暂停倒计时，离开后自动恢复，带进度条显示剩余时间

📊 优先级队列

支持低/普通/高/紧急四个优先级，高优先级通知停留时间更长

👆 手动关闭

点击关闭按钮或整个通知区域即可手动关闭，带有平滑的滑出动画

📱 响应式设计

适配移动端和桌面端，在小屏幕上自动调整宽度和间距

API 参考

// 基础用法
notify.show({
  type: 'success',           // 类型: success | error | warning | info
  title: '标题',            // 通知标题
  message: '详细内容',      // 通知内容
  priority: 'normal',       // 优先级: low | normal | high | critical
  duration: 5000,             // 自定义停留时间(毫秒)
  showProgress: true,         // 是否显示进度条
  onClose: () => {},          // 关闭回调
  onClick: () => {}           // 点击回调
});

// 快捷方法
notify.success(message, title, options);
notify.error(message, title, options);
notify.warning(message, title, options);
notify.info(message, title, options);

// 关闭通知
notify.close(notificationId);   // 关闭指定通知
notify.closeAll();               // 关闭所有通知
